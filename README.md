# Description

Komubot-dashboard is a web project for administrators to manage Komubot on Discord. The code is built using React TypeScript.

# Getting Started

## Installation

```bash
$ npm install
```

## Env

Add next .env config in root

```bash
REACT_APP_GOOGLE_CLIENT_ID=
REACT_APP_API_URL=
```

## Run

```bash
$ npm start
```

## Test

```bash
$ npm test
```
